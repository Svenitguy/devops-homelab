# devops-homelab


## 📌 Doel
Dit project is een DevOps labo waarin ik stap voor stap een volledige CI/CD pipeline bouw met Docker, GitHub Actions en Terraform (Azure).

---

## 🧰 Technologieën
- Python (Flask)
- Docker
- Git & GitHub
- GitHub Actions (CI/CD)
- Terraform (Azure infrastructure)

---

## 🏗️ Architectuur (fase 1)
Local development → Docker container → GitHub repo → CI/CD pipeline (later)

---

## 🚀 Hoe runnen (later)
```bash
docker build -t devops-homelab .
docker run -p 5000:5000 devops-homelab
